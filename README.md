## 📦 Quick Start

<details>
<summary><b>🛠️ Installation Guide</b></summary>

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/careeros.git
   ```

2. **Install Dependencies**
   ```bash
   npm install
   ```

3. **Environment Setup**
   ```bash
   # .env
   PORT=5000
   MONGO_URI=your_uri
   JWT_SECRET=your_secret
   AI_API_KEY=your_key
   ```

4. **Launch Application**
   ```bash
   npm run dev
   ```

</details>
